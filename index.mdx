## Core Capabilities

<Accordion title="Real-time Order Synchronization">
  Built on Supabase Realtime, Lurwis CRM provides instant updates when new orders arrive. The dashboard automatically refreshes with new orders, status changes, and cancellations without manual intervention.
</Accordion>

<Accordion title="Kanban Board Management">
  Visualize and manage your order workflow with a drag-and-drop Kanban board. Move orders through states: **pendiente** (pending) → **cocina** (kitchen) → **entregar** (delivery) → **completado** (completed).
</Accordion>

<Accordion title="Analytics & Reporting">
  Track business metrics including total revenue, average ticket value, success rate, top dishes, payment method distribution, delivery vs pickup ratio, and peak hours visualization.
</Accordion>

<Accordion title="Order History">
  Search and filter completed and cancelled orders by date range and payment method. Export data for accounting and business analysis.
</Accordion>

## Technology Stack

Lurwis CRM is built with modern web technologies:

- **Frontend**: React 19 with Vite
- **Styling**: Tailwind CSS
- **Backend**: Supabase (PostgreSQL + Realtime)
- **Authentication**: Supabase Auth
- **Routing**: React Router v7
- **Integration**: Webhooks (N8N/Make/Zapier)
